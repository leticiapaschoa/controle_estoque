# Aplicação Web para controle de estoque
- [x] SPA
- [x] PWA
- [x] Angular
- [x] Firebase

### Setup ambiente:
Comandos básicos para iniciar sua aplicação:
```
  - Instalar o angular:
  npm install -g @angular/cli
  
  - Criar um workspace:
  ng new my-app
  
  - Rodar a aplicação no servidor:
  ng serve --open
```
